## O que é programação?

Programação é o processo de escrever instruções para que um computador execute tarefas e resolva problemas.

Assim como humanos usam idiomas para se comunicar, computadores utilizam **linguagens de programação** para entender instruções.

Cada linguagem possui características próprias e pode ser mais adequada para determinados tipos de problema.

### Exemplos de linguagens

- **Python** → muito usado em ciência de dados e automação. Linguagem simples e versátil.
- **Java** → comum em sistemas corporativos e aplicações robustas.
- **JavaScript** → amplamente usado no desenvolvimento de sites e aplicações web.

---

## Lógica de programação

Antes de aprender uma linguagem específica, é fundamental aprender **lógica de programação**.

A lógica permite:

- analisar um problema
- dividir o problema em passos
- criar uma sequência de instruções

Essa sequência de passos é chamada de **algoritmo**.

Um algoritmo pode ser implementado em qualquer linguagem de programação.

## Áreas de atuação em programação

Algumas áreas comuns na tecnologia são:

- **Front-end** → desenvolvimento da interface de sites e aplicações.
- **Back-end** → desenvolvimento da lógica e funcionamento interno das aplicações.
- **Desenvolvimento Mobile** → criação de aplicativos para celulares.
- **Desenvolvimento de jogos**
- **Segurança da informação**
- **Inteligência Artificial**

## Mercado de trabalho e desenvolvimento contínuo

Para se destacar no mercado de tecnologia é importante:

- demonstrar evolução e projetos realizados
- compartilhar conhecimento e aprendizado

### GitHub

O **GitHub** é uma plataforma onde podemos publicar projetos e construir um portfólio.

Recrutadores frequentemente analisam o GitHub para entender:

- nível técnico
- qualidade do código
- projetos desenvolvidos
- evolução ao longo do tempo

### LinkedIn

Também é importante manter um perfil ativo no LinkedIn:

- conectar-se com profissionais da área
- seguir empresas
- participar de comunidades
- compartilhar aprendizados

Manter presença online ajuda a aumentar visibilidade e oportunidades.
